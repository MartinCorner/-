# pomodoro-clock
时间管理钟/桌面时钟/番茄工作法

因为当时正准备买一个桌面时钟，同时有想要用番茄钟来约束自己的工作时间，于是这个二合一的小工具就被我想出来了。
运行设备是一台3寸屏的安卓手机，用浏览器访问服务器上的html页面
